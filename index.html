Welcome to Simutrans on github

Simutrans is a transport simulation game. The game is open source under the artistic license 2.0. The game simulates transport of goods and passengers with multiple modes of transport (viz Rail, Road, Aeroplane, Ship). The game world is seen in an isometric view.

Several modular 'pak-sets', containing graphical game content and game data are also available, several also under open licenses.

Simutrans repositories at github

  • Simutrans git mirror: origianl SVN on svn://tron.homeunix.org/simutrans/simutrans/ maintained by Ansgar Burchardt
  • Simutrans-Experimental: official development repository, maintained by James Petts
  • pak128.britain: mirror of SVN repository on sourceforge (see below), maintained by Nathanael Nerode

  • pak128: mirror of the SVN repository on sourceforge (see below), maintained by the pak128 team (@pak128svn)

pak128

pak128 is a so called pak-set for simutrans, it contains graphics for world tiles and vehicles, as well as game data for these objects. The pak-set is under Artistic License 2.0. The repository on github is a mirror of the main SVN repository on sourceforge http://sourceforge.net/projects/simutrans/files/pak128/

The maintainers and main developers of the pak-set can be contacted at their forum thread: http://forum.simutrans.com/index.php?board=26.0 Requests and bug reports posted here will be forwarded however (for the foreseeable future at least, state 2012).
